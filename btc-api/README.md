# Bitcoin Node API — Umbrel App

Lightweight REST API for your local Bitcoin node + Mempool. Built with FastAPI, only 4 dependencies.

## Dependencies

| Package  | Purpose                        |
|----------|--------------------------------|
| fastapi  | Web framework                  |
| uvicorn  | ASGI server                    |
| bit      | Bitcoin key/transaction utils  |
| httpx    | Async HTTP client for Mempool  |

## Endpoints

### Node Info
| Method | Endpoint        | Description                      |
|--------|-----------------|----------------------------------|
| GET    | `/info`         | Node status, sync, connections   |
| GET    | `/blockheight`  | Current block height             |
| GET    | `/health`       | Health check (node + mempool)    |

### Fees (via Mempool)
| Method | Endpoint              | Description              |
|--------|-----------------------|--------------------------|
| GET    | `/fees/recommended`   | Recommended fee rates    |
| GET    | `/fees/mempool`       | Fee distribution         |

### Address
| Method | Endpoint                   | Description              |
|--------|----------------------------|--------------------------|
| GET    | `/address/{addr}`          | Balance & tx count       |
| GET    | `/address/{addr}/utxos`    | Unspent outputs          |
| GET    | `/address/{addr}/txs`      | Transaction history      |

### Transactions
| Method | Endpoint          | Description                          |
|--------|-------------------|--------------------------------------|
| GET    | `/tx/{txid}`      | Transaction details                  |
| GET    | `/tx/{txid}/hex`  | Raw transaction hex                  |
| GET    | `/tx/{txid}/rsz`  | Extract R, S, Z, pubkey per input    |
| POST   | `/tx/broadcast`   | Broadcast raw tx `{"hex": "..."}`    |

### Sweep
| Method | Endpoint       | Description                          |
|--------|----------------|--------------------------------------|
| POST   | `/sweep/wif`   | Sweep from WIF key `{"key": "..."}`  |
| POST   | `/sweep/hex`   | Sweep from hex key `{"key": "..."}`  |

### Mempool
| Method | Endpoint           | Description              |
|--------|--------------------|--------------------------|
| GET    | `/mempool`         | Mempool stats from node  |
| GET    | `/mempool/recent`  | Recent mempool txs       |

## RSZ Endpoint

`GET /tx/{txid}/rsz` parses the raw transaction and extracts ECDSA signature components for each input:

```json
{
  "txid": "abc123...",
  "inputs": [
    {
      "vin": 0,
      "R": "a1b2c3...",
      "S": "d4e5f6...",
      "Z": "789abc...",
      "pubkey": "04abcd...",
      "sighash_type": 1
    }
  ]
}
```

**Supported input types:**
- P2PKH (legacy) — full R, S, Z extraction
- P2WPKH (segwit v0) — R, S extracted; Z shows `requires_utxo_amount` (BIP143 needs the input value which isn't in the raw tx)
- Multisig / Taproot — not currently supported

## Environment Variables

| Variable              | Default         | Description                 |
|-----------------------|-----------------|-----------------------------|
| `BITCOIN_RPC_USER`    | `umbrel`        | Node RPC username           |
| `BITCOIN_RPC_PASS`    | —               | Node RPC password           |
| `BITCOIN_RPC_HOST`    | `bitcoin`       | Node hostname/IP            |
| `BITCOIN_RPC_PORT`    | `8332`          | Node RPC port               |
| `MEMPOOL_HOST`        | `mempool_api`   | Mempool API hostname/IP     |
| `MEMPOOL_PORT`        | `8999`          | Mempool API port            |
| `SWEEP_ADDRESS`       | —               | Destination for sweep calls |

## Install on Umbrel

Copy this folder to your Umbrel app directory and install, or run manually:

```bash
docker-compose up -d
```

API docs available at `http://umbrel.local:8000/docs` (auto-generated by FastAPI).
