Cripto Suite — herramientas Bitcoin offline

Cada proyecto open-source corre tal cual upstream, en su propio <iframe sandbox>. El aislamiento entre herramientas se mantiene porque cada una corre en su propio subdirectorio y no comparte estado con las otras.

Notas de seguridad

• RNG: signature-tool y coinbin usan librerías de ~2013 para generación de claves. Para generar claves nuevas preferir BIP39, WarpWallet, bitaddress o mnemonic-offline-tool que usan crypto.getRandomValues. Para firmar/verificar/derivar con claves existentes el riesgo es menor.

• Offline: la garantía es operacional. coinbin puede hacer llamadas de red (broadcast/fees) que fallarán sin conexión. Usar siempre sin conexión / air-gapped para material de claves.

• Los iframes file:// pueden bloquear subrecursos en Chrome: servir siempre vía HTTP (Docker o python -m http.server).
